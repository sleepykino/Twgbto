# 小说续写工具

基于Vue3 + TypeScript开发的智能小说续写工具，支持调用GLM和DeepSeek等大模型API进行AI辅助创作。新增角色卡和世界书功能，让AI续写更加智能和个性化。

## 功能特点

### 核心功能
- **智能续写**: 支持GLM和DeepSeek系列大模型API，可根据上下文和提示词智能续写小说内容
- **角色卡系统**: 类似SillyTavern的角色卡功能，可以创建和管理角色信息，包括性格、外貌、背景、说话风格等
- **世界书功能**: 支持创建世界设定条目，通过关键词触发，自动注入相关设定信息
- **章节管理**: 树形结构章节管理，支持添加、删除、移动、展开/折叠章节
- **文风选择**: 内置多种写作风格（默认、快节奏爽文、言情细腻、悬疑智斗、诗意文学），支持自定义提示词
- **富文本编辑**: 基于Quill的富文本编辑器，支持标题、加粗、斜体、下划线、删除线、颜色、背景色、列表、缩进、链接、图片等格式
- **本地存储**: 自动保存章节内容、角色卡、世界书、设置和文风偏好到本地存储
- **导出功能**: 支持将所有章节导出为TXT文件

### 角色卡功能
- **多角色管理**: 创建和管理多个角色，每个角色都有独立的属性
- **详细角色信息**: 支持设置角色名称、描述、性格、外貌、背景故事、说话风格、人物关系、备注等
- **标签系统**: 为角色添加标签，方便分类和查找
- **启用/禁用**: 可以灵活控制哪些角色参与续写
- **导入导出**: 支持角色数据的导入导出（JSON格式）
- **快速复制**: 一键复制角色卡，方便创建相似角色

### 世界书功能
- **多世界书管理**: 创建多个世界书，每个世界书可以包含不同的设定
- **分组管理**: 将世界书条目分组管理，便于组织不同类型的设定
- **关键词触发**: 通过关键词自动触发相关设定，支持正则表达式
- **优先级控制**: 设置条目优先级，控制注入顺序
- **灵活配置**: 支持设置扫描深度、令牌预算、递归扫描等参数
- **位置控制**: 控制条目注入的位置（角色前/后、示例前/后）
- **导入导出**: 支持世界书的导入导出（JSON格式）

### 界面设计
- **可折叠侧边栏**: 左侧设置面板可折叠，节省编辑空间
- **响应式布局**: 适配不同屏幕尺寸
- **实时字数统计**: 显示当前章节字数
- **直观的操作界面**: Element Plus UI组件，简洁美观
- **智能提示**: 续写时显示启用的角色数量和匹配的世界书条目

## 技术栈

- **框架**: Vue 3 (Composition API)
- **语言**: TypeScript
- **UI框架**: Element Plus
- **富文本编辑器**: Quill
- **状态管理**: Pinia
- **HTTP客户端**: Axios
- **构建工具**: Vite
- **代码规范**: ESLint + Prettier

## 使用指南

### 1. 设置API密钥
- 在左侧设置面板中展开对应的模型（GLM或DeepSeek）
- 输入您的API密钥
- 调整温度参数（0-2之间，数值越高创造性越强）

### 2. 创建角色卡
- 点击"角色卡"展开角色管理面板
- 点击"添加角色"创建新角色
- 填写角色信息：
  - **基本信息**: 名称、描述、头像URL、标签
  - **性格特征**: 性格、说话风格
  - **背景故事**: 外貌描述、背景故事
  - **关系备注**: 人物关系、其他备注
- 使用开关控制角色是否参与续写
- 可以复制、删除、导入导出角色

### 3. 创建世界书
- 点击"世界书"展开世界书管理面板
- 点击"新建世界书"创建世界书
- 在世界书中创建分组（如：地理设定、历史背景、魔法体系等）
- 在分组中添加条目：
  - **键名**: 条目的唯一标识
  - **关键词**: 触发该条目的关键词（支持多个）
  - **内容**: 条目的详细内容
  - **优先级**: 控制注入顺序（数值越大越优先）
  - **位置**: 控制注入位置
  - **高级选项**: 区分大小写、使用正则表达式
- 当续写内容包含关键词时，相关条目会自动注入到AI提示词中

### 4. 章节管理
- 点击"章节管理"展开章节列表
- 点击"添加章节"按钮创建新章节
- 右键点击章节可进行删除、移动等操作
- 点击章节标题切换到对应章节

### 5. 文风选择
- 在"文风选择"中选择预设的写作风格
- 或在提示词框中输入自定义提示词来控制AI续写风格

### 6. 编写小说
- 在右侧编辑器中输入您的小说内容
- 使用工具栏进行格式设置（标题、加粗、斜体、颜色等）
- 实时查看字数统计

### 7. 续写小说
- 点击"续写"按钮
- 在弹出的对话框中：
  - 输入续写提示词
  - 选择模型和续写长度
  - 查看启用的角色数量
  - 查看匹配的世界书条目
  - 选择是否使用角色卡和世界书
- 点击"开始续写"等待AI生成内容
- 续写内容会自动插入到编辑器末尾

### 8. 保存和导出
- 点击"保存"按钮将当前章节内容保存到本地存储
- 点击"导出TXT"按钮将所有章节导出为TXT文件

## 配置说明

### API密钥配置
- GLM API密钥: 在智谱AI开放平台获取
- DeepSeek API密钥: 在DeepSeek开放平台获取

### 温度参数
- 范围: 0-2
- 低值（0-0.3）: 输出更确定、保守
- 中值（0.3-0.7）: 平衡创造性和一致性
- 高值（0.7-2）: 输出更有创造性、多样性

### 写作风格
- 默认: 和原文保持一致
- 快节奏爽文: 情节紧凑，爽点密集，语言简练有力
- 言情细腻: 注重情感描写，语言优美，情感丰富
- 悬疑智斗: 设置悬念，逻辑严密，注重推理过程
- 诗意文学: 语言优美，意境深远，富有文学性

### 世界书配置
- **扫描深度**: 控制扫描文本的深度（默认2）
- **令牌预算**: 世界书条目的最大令牌数（默认2048）
- **递归扫描**: 是否递归扫描匹配的条目

## 角色卡与世界书的工作原理

### 角色卡注入
当启用角色卡时，AI会在续写时收到以下格式的角色信息：

```
=== 角色信息 ===
【角色：张三】
描述：主角，年轻剑客
性格：勇敢、正直、有些冲动
外貌：身材高大，剑眉星目
背景：出身武林世家，自幼习武
说话风格：直爽豪迈，不拘小节
人物关系：与李四是好友
备注：擅长剑法，特别是"流云剑法"
=== 角色信息结束 ===
```

### 世界书触发
当文本中出现关键词时，相关的世界书条目会被注入：

```
=== 世界设定 ===
【流云剑法】
张家的家传剑法，以快速灵活著称，共有三十六式，其中最厉害的是"云龙九现"。

【武林盟】
江湖上最大的组织，由各大门派组成，负责维护江湖秩序。
=== 世界设定结束 ===
```

## 数据存储

所有数据存储在浏览器的localStorage中，包括：
- 章节内容和结构
- 角色卡数据
- 世界书数据
- API密钥和模型设置
- 文风偏好
- 当前选中章节
